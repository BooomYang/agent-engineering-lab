# Framework Roadmap

这里记录未来可能框架化、产品化、自动化的能力。

本目录不急于实现。只有当某类问题在多个实践中反复出现，并且已经有清晰模板或 eval 支撑时，才考虑升级为框架能力。

## Candidate Capability Areas

- agent design brief generator。
- tool spec linter。
- structured output schema reviewer。
- guardrail/approval policy generator。
- trace-to-eval case extractor。
- eval regression runner。
- agent production readiness checker。
- practice retro summarizer。

当前 backlog 见 [backlog.md](backlog.md)。
