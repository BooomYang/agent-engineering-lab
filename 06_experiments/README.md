# Experiments

这里存放小实验、对照验证、失败样本和待验证假设。

## Experiment Types

- prompt/instructions 对照。
- tool description 对照。
- structured output schema 对照。
- single agent vs multi-agent 对照。
- handoff vs agent-as-tool 对照。
- state strategy 对照。
- guardrail 阻断策略对照。
- eval scorer 对照。

## Experiment Template

```md
# Experiment: <name>

## Hypothesis

## Setup

## Variables

## Cases

## Results

## Interpretation

## Decision

## Follow-up
```

## Rule

实验结论必须保留上下文条件。不要把一个小实验结论直接推广成通用原则。
