# Sources

这里存放 agent engineering 的资料来源、精读清单和文章笔记。

## Source Types

- `official`：OpenAI 官方文档、官方 cookbook、官方博客。
- `external`：第三方技术文章、论文、开源项目、访谈、演讲。
- `practice`：自己的项目实践、调试过程、复盘。
- `hypothesis`：暂时没有足够证据，但值得验证的判断。

## Intake Rules

1. 新来源先登记到 [sources.md](sources.md)。
2. 外部文章不要直接变成规则，先按 [article-note-template.md](article-note-template.md) 拆成事实、观点、推断、待验证假设。
3. 和 OpenAI 产品能力有关的内容，需要回到官方文档校准。
4. 能落地的结论再分流到方法论、实践、模板、案例或实验目录。
